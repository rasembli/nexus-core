# Security Policy

Nexus Core is a cryptography-forward project. Responsible disclosure of
security issues is welcome and important.

## Reporting a vulnerability

Email aubrey@martechadvisor.io with a description of the issue and steps
to reproduce. Expect an acknowledgement within 72 hours.

Do not open a public issue for security-sensitive reports.

## Supported versions

Nexus Core is in early development. Security fixes are applied to the
current main branch. Version-specific support will be published once
the framework reaches v1.0.
